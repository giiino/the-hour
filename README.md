# The Hour 新聞網
製作於2021年大學練手作品，為獨立完成的網站 [原Git](https://github.com/giiino/news-web)，2022年重建repo

之後有時間陸續添加新功能 :
- [x] 留言系統
- [x] 後端API開發使用Express
- [x] 部屬到Vercel
- [x] 資料庫使用db4free
- [x] 導入MUI優化界面
- [ ] 使用Vercel cron job定期更新新聞(原本預計，但hobby模式下不讓我這麼做)
- [ ] 個人最愛文章

## 技術棧
### 前端
- React Class Component
- react-router-dom
- react-redux
- redux-thunk
- redux-immutable
- react-icons
- styled-components

### 後端
- Express
- firebase
- mysql
